## Overview
Engineered **full-stack platform** using React.js, Node.js, and Python, serving **15+ lawyers** with AI-powered document automation and LaTeX compilation for professional legal document generation.

## Key Features
- Implemented **Firebase Authentication** with JWT token validation for role-based access control
- Built **RESTful APIs** handling document CRUD operations with Firestore for real-time synchronization
- Integrated **LLM for legal text generation** with LaTeX compilation, enabling AI-driven workflows for professional legal document creation
- Reduced manual document preparation time by **97%**
