## Overview
Developed **dual-sided platform** connecting freelancers and employers with separate dashboards for job search, skill-based filtering, application tracking, task posting, and hiring management.

## Key Features
- Implemented **MVC architecture** with Express.js routing, middleware authentication, MongoDB schemas for structured data
- Integrated **Firebase Realtime Database** for instant notifications
- Separate dashboards for freelancers and employers with comprehensive feature sets

## Technologies
React.js, Node.js, Agentic AI, Python, Flask, Firebase, MongoDB, MVC, JWT, Express.js

**Note:** This is a startup project I co-founded and developed.
